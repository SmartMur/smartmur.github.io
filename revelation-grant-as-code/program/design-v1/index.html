    Program Design: Version 1¶

    Based on community assessment findings - March 10, 2026

    🎯 Design Principles¶

    Principle Source
    Community-led Advisory committee (51% immigrant mothers)
    Barrier-first Address childcare, devices, confidence before curriculum
    Job-connected Employer partnerships from day one
    Peer-powered Learning happens in community
    Flexible Designed for mothers' actual schedules
    Transparent Open source, publicly documented

    👥 Participant Profile¶

    Characteristic Target
    Who Immigrant mothers (any country)
    Children Any age (childcare provided)
    Time in Canada 0-10 years
    Current work Unemployed or underemployed
    Tech experience Any level, including none
    Language All languages (interpretation provided)
    Income Low-income (program free)

    Cohort size: 15-20 women

    📅 Program Structure¶

    Duration: 6 months¶

    Phase Duration Focus
    Phase 1: Foundations Months 1-2 Digital literacy, confidence building, peer connections
    Phase 2: Skills Months 3-4 Technical skills (web development, data, or design)
    Phase 3: Job Readiness Month 5 Resume, interviews, employer connections
    Phase 4: Placement Month 6 Job search support, mentorship, alumni network

    🗓️ Weekly Schedule¶

    Primary Option: Weekday Mornings - Tuesday, Wednesday, Thursday - 9:30am - 2:30pm (5 hours) - Includes: instruction, practice, lunch, peer time

    Secondary Option: Saturdays - Saturday mornings - 9:30am - 1:30pm (4 hours) - For women whose partners work weekends

    Why this schedule? 84% of participants preferred weekday mornings; only 8% preferred evenings.

    🧩 Core Components¶

    1. Childcare (On-site)¶

    Detail Description
    Ages 0-5 years (separate room), 6-12 years (homework help)
    Staff Qualified early childhood educators
    Ratio 1:3 for infants, 1:5 for toddlers, 1:8 for school-age
    Schedule Same as program hours
    Cost Free to participants

    Source: 84% identified childcare as primary barrier

    2. Device Provision¶

    Item Description
    Laptop Refurbished or new laptop for each participant
    Ownership Yours to keep after program completion
    Hotspot Mobile hotspot or internet subsidy
    Support Basic tech support throughout program

    Source: 62% lack adequate devices

    3. Peer Cohorts¶

    Element Description
    Cohort size 15-20 women start and finish together
    Peer circles Small groups (4-5) within cohort
    WhatsApp groups Ongoing connection between sessions
    Social events Monthly community gatherings

    Source: 58% want peer learning; 48% experience isolation

    4. Mentorship¶

    Element Description
    Mentors Women in tech (immigrant background preferred)
    Match 1 mentor per 3-4 participants
    Frequency Monthly group mentoring + optional 1:1
    Focus Career guidance, confidence, network

    Source: 72% confidence gap

    5. Employer Partnerships¶

    Element Description
    Partners 5-10 committed employers
    Involvement Guest talks, site visits, job shadows
    Hiring Pipeline for program graduates
    Support Ongoing support for employers and graduates

    Source: 86% want job placement, not just training

    6. Language Support¶

    Element Description
    Interpretation Available for all sessions
    Materials Key documents in multiple languages
    Facilitators Bilingual where possible
    Language learning Integrated English/French support

    Source: 56% language barrier

    📚 Curriculum Overview¶

    Phase 1: Foundations (Weeks 1-8)¶

    Week Focus Output
    1 Welcome, community building, tech setup Cohort connected
    2 Digital basics: files, folders, internet Comfort with computer
    3 Online safety, passwords, privacy Safe practices
    4 Email, communication tools Professional email
    5 Introduction to web How internet works
    6 Career exploration Interest identified
    7 Confidence building Personal story
    8 Phase 1 celebration Portfolio start

    Phase 2: Skills (Weeks 9-16)¶

    Participants choose one track

    Track A: Web Development - HTML/CSS basics - Building simple websites - JavaScript fundamentals - Project: Personal portfolio site

    Track B: Data - Spreadsheets fundamentals - Data visualization - Introduction to SQL - Project: Data analysis portfolio

    Track C: Digital Design - Canva and design basics - Social media content - Introduction to UX - Project: Design portfolio

    Phase 3: Job Readiness (Weeks 17-20)¶

    Week Focus
    17 Resume and LinkedIn workshop
    18 Interview practice (with employers)
    19 Job search strategies
    20 Networking event with employers

    Phase 4: Placement (Weeks 21-24)¶

    • Supported job applications
    • Employer introductions
    • Interview coaching
    • Job offer support
    • Alumni network entry

    📊 Success Metrics¶

    Outcome Target Measurement
    Completion rate 80% Attendance records
    Job placement 70% within 6 months Follow-up surveys
    Income increase $15,000 average Pre/post income
    Confidence gain 50% increase Pre/post survey
    Device ownership 100% keep laptop Tracking
    Alumni engaged 50% in network Participation

    🔒 Scope Lock Decisions (Q2 2026)¶

    The advisory pre-read and operations review on March 10, 2026 produced a scoped 90-day build window:

    1. Launch one pilot focus-group sequence before adding additional tracks.
    2. Keep one primary schedule: weekday mornings, with one evening overflow slot.
    3. Treat childcare, interpretation, and transportation support as baseline budget lines.
    4. Prioritize web and workforce readiness modules before adding advanced specializations.
    5. Gate grant submissions through a shared evaluation and readiness checklist.
